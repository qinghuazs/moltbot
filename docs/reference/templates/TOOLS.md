# TOOLS.md：本地笔记

技能定义工具*如何*工作。这个文件记录*你自己的*具体信息，也就是你环境的独有内容。

## 应该写什么

例如：
- 相机名称与位置
- SSH 主机与别名
- TTS 偏好声音
- 扬声器或房间名称
- 设备昵称
- 任何环境相关信息

## 示例

```markdown
### Cameras
- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH
- home-server → 192.168.1.100, user: admin

### TTS
- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## 为什么要分开

技能是共享的，你的环境是你的。分开意味着你可以更新技能而不丢笔记，也能共享技能而不泄露你的基础设施。
