# `moltbot setup`

初始化 `~/.clawdbot/moltbot.json` 与代理工作区。

相关：
- 快速开始：[Getting started](/start/getting-started)
- 向导：[Onboarding](/start/onboarding)

## 示例

```bash
moltbot setup
moltbot setup --workspace ~/clawd
```

通过 setup 启动向导：

```bash
moltbot setup --wizard
```
