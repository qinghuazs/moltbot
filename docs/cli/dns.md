# `moltbot dns`

广域发现的 DNS 助手（Tailscale + CoreDNS）。目前专注于 macOS + Homebrew CoreDNS。

相关：
- 网关发现：[发现](/gateway/discovery)
- 广域发现配置：[配置](/gateway/configuration)

## 设置

```bash
moltbot dns setup
moltbot dns setup --apply
```
